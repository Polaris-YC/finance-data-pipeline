### Overview  
This repository contains a Python-based toolkit for collecting, storing, and maintaining financial time-series data using a local SQL database.  
### Key features  
1. Automated data collection from external api  
2. managing structured data with SQL backend
3. handling large panel datasets
### Structure
`database_manager.py`: the core codes  
`docs/userguide`: documentation of available functions and parameters  
### Note
This repository is provided as a code sample to demonstrate my ability to work with Python on data projects involving databases and automated data processing. The code was originally developed for financial data analysis and is not intended to be a finished program package.
